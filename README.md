# Java Calculator

A simple calculator built in Java using Scanner and if/else logic.

## Features
- Addition
- Subtraction
- Multiplication
- Division

## How to run
Compile and run:

javac ProjectAlpha.java
java ProjectAlpha.

Remember: Type the First number then click "Enter" on your keyboard
and then add the other number and then the -, +, /, and *
